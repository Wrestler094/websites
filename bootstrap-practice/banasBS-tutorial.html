Table notes:

  • all table classes are in the actual table element, not the div.
  • The table above is in a container with .col-md-6 AND .col-md-offset-6 which pushes the table over 6 columns.
    • At md or bigger, the table is 6 columns over. At the sm and xs sizes, it takes up the entire 12 columns.
  • class "table-bordered" puts the border around the table.
  • class "table-striped" gives it that white and grey alternating look
  • class "table-hover" highlights the row when hovering over an item in the table
  • For the "Best Baseball Players" header, use 'colspan="4"' to indicate 4 columns
  • We can give items the class "text-center" to center the text

Carousel Notes

  • Here is the div and element breakdown for the carousel:
    • Div with id (specific to carousel), classes ('carousel', 'slide'), and data-ride ('carousel'); I don't know what data-ride is.
      • Ordered List which is the number of slides in your carousel. They each have special properties like 'data-target' which is equal to the id of your carousel div above AND 'data-slide-to' equal to the slide number. Slide 0 is the first slide. You can also add the "active" class to slide 0.
      • Div with class="carousel-inner"
        • Div with class="item"
          • The image. In the above example, it is a Div class="slide#" => slide1, slide2, etc. are class names given by Derek.
          • Div class="carousel-caption"
      • Links (<a>) with class="carousel-control" and either left or right classes.
  • In the above example, slide1, slide2 and slide3 classes are used by Derek, they aren't Bootstrap classes. They contain the background images for my carousel.
